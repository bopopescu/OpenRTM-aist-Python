マネージャの初期化 

マネージャを初期化する static 関数。 マネージャをコマンドライン引数を与えて初期化する。 マネージャを使用する場合は、必ずこの初期化メンバ関数 init() を 呼ばなければならない。 マネージャのインスタンスを取得する方法として、init(), \doxyref{instance()}{p.}{classsource__py_1_1_manager_1_1_manager_3bcc7c212b5d936cfd55fa669ed61735} の 2つの static 関数が用意されているが、初期化はinit()でしか行われないため、 \doxyref{Manager}{p.}{classsource__py_1_1_manager_1_1_manager} の生存期間の一番最初にはinit()を呼ぶ必要がある。

※マネージャの初期化処理\begin{itemize}
\item initManager: 引数処理、configファイルの読み込み、サブシステム初期化\item initLogger: Logger初期化\item initORB: ORB 初期化\item initNaming: NamingService 初期化\item initExecutionContext: ExecutionContext factory 初期化\item initTimer: Timer 初期化\end{itemize}
