OutPort 基底クラス 


OutPort の実装である OutPort<T> の基底クラス。

OutPortBase と PublisherBase は一種の Observer パターンを形成している。 OutPortBase の attach(), detach(), notify() および PublisherBase の push() は Observer パターンに関連したメソッドである。
